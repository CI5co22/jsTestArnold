Pregunta 1

¿Cuál es la diferencia entre var, let y const en JavaScript? Proporciona un ejemplo de cada uno.

Una de las diferencias es el scope de cada uno, siendo su accesibilidad distinta. Ademas, con var se puede redeclarar y reasignar, mientras que con let solo se puede reasignar mas no redeclarar y const no se puede hacer ninguna de estas

Ejemplos:
var nombre = "Arnold"
let pais = "Guatemala"
const PI = 3.1416
